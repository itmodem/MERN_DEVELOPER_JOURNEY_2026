# 📘 **DAY 1 — STUDY NOTES**

## **HTML Basics: Tags, Elements & Structure**


# 🟦 **1. SUMMARY (High-Clarity Overview)**

HTML (HyperText Markup Language) defines the **structure** of every webpage.
It tells the browser **what each part of the page means** — headings, paragraphs, images, sections, navigation, etc.

HTML is **not a programming language**.
It is a **markup language** used to create content layout.

**Hindi (logic):**
*HTML ko ghar ki foundation samjho. Backbone strong hoga toh upar React, Tailwind, JS sab smooth chalega.*

---

# 🟧 **2. KEY CONCEPTS**

These are the core ideas you MUST understand on Day-1.

---

## **1️⃣ Tags**

A tag is a keyword inside angle brackets:

```
<h1>
<img>
<p>
```

Types:

* **Opening tag** → `<p>`
* **Closing tag** → `</p>`
* **Self-closing tag** → `<img />`, `<br />`

**Hindi:**
*Tag = structure ka starting point.*

---

## **2️⃣ Elements**

An element is the **complete unit**:

```
<p>Hello</p>
```

Element = opening tag + content + closing tag.

**Hindi:**
*Element = poora block. Jaise room ka pura structure.*

---

## **3️⃣ Attributes**

Extra information added to a tag.

Example:

```html
<img src="image.jpg" alt="My photo" />
```

Attributes give **functionality** to tags.

---

## **4️⃣ Semantic HTML**

Semantic tags tell the browser **meaning** of the content.

🏷 Semantic Examples:

* `<header>`
* `<nav>`
* `<main>`
* `<section>`
* `<article>`
* `<footer>`

**Why important?**

* Better SEO
* Cleaner structure
* Easier accessibility

**Hindi:**
*Semantic tags webpage ko dimaag dete hain.*

---

## **5️⃣ HTML Skeleton (Document Structure)**

This is the minimum structure of every webpage:

```html
<!DOCTYPE html>
<html lang="en">

<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <title>My First HTML Page</title>
</head>

<body>
  <!-- Visible content will go here -->
</body>

</html>
```

**Breakdown:**

* `<!DOCTYPE html>` → tells browser to use modern HTML
* `<html>` → root
* `<head>` → meta data, title, SEO
* `<body>` → visible page content

# 🟩 **3. CODE EXAMPLES (Copy-Paste Ready)**

---

## **Example 1 — Basic Webpage Structure**

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <title>Day 1 - HTML Structure</title>
</head>

<body>
  <header>
    <h1>Welcome to My Webpage</h1>
  </header>

  <main>
    <section>
      <p>This is my first paragraph.</p>
      <p>This is the second one.</p>
    </section>

    <img src="profile.jpg" alt="Profile Image" />
  </main>

  <footer>
    <p>© 2026 Your Name</p>
  </footer>
</body>
</html>
```

---

## **Example 2 — Semantic Layout**

```html

    <p>This is an article explaining semantic tags.</p>
  </article>
</main>

<footer>
  <p>Contact: you@example.com</p>
</footer>
```
