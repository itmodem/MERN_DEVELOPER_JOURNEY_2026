# 🎬 **SHORT SCRIPT — “var / let / const in 30 Seconds”**

Listen carefully…

In JavaScript, we have **three ways** to make variables:

### **1️⃣ var**

Old style.
Function-scoped.
Can be re-declared.
(Hindi: bahut bugs create karta hai… avoid it.)

### **2️⃣ let**

Modern, clean.
Block-scoped.
Reassign allowed, re-declare not allowed.
Best for values that change — like counters, inputs, loops.

### **3️⃣ const**

Constant values.
Block-scoped.
Cannot reassign.
Perfect for configs, arrays, objects.
(Hindi: const use karo jab value change nahi honi chahiye.)

### **Simple Rule:**

👉 **Use let when value changes**
👉 **Use const when value stays same**
👉 **Avoid var**

That’s it — clean, simple, professional JavaScript.
